# infinitynikkiensou
このプロジェクトは、Windows 環境で動作する自動音楽演奏ツールです。指定された楽譜に基づいて、ゲーム内で自動的に音楽を演奏します。

## 特徴
- 簡単なテキスト形式の楽譜をサポート
- 2 オクターブの音域（C3 ～ B4）に対応
- 長音や休符をサポート
- 自動で依存関係をインストール

## 必要な環境
- Windows 10 以上
- Python 3.7 以上
